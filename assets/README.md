Placez les fichiers médias fournis dans les dossiers suivants :

- `assets/images/` : logos, photos, visuels de projets (formats recommandés : `.jpg`, `.png`, `.webp`, `.svg`)
- `assets/icons/` : pictogrammes du workflow et autres icônes (`.svg` de préférence)
- `assets/fonts/` : polices web (`.woff2`) telles que `RocaOne-Black.woff2` et `Raleway-VariableFont_wght.woff2`

Mettez à jour les noms de fichiers dans le code si nécessaire pour correspondre aux ressources ajoutées.
